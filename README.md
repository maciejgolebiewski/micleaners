# MICleaners


## Usage

1. Clone this repository

2. Change directory to the application folder

        $ cd micleaners

3. Install the dependencies

        $ npm install

4. Run the gulp command

        $ gulp

5. Using a browser, go to `http://localhost:3000` to load the initial page
